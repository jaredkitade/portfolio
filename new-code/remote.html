Remotely Remote

Web Design+Development

In 2016, I was hired to design and code a website to promote CCA’s annual Industrial Design Senior Show. The show’s theme changes each year, and this year it was interconnectivity and human interaction. The committee running the show wanted to have a site that was unique and reflected the work and personalities of the graduating seniors.

I came up with the idea to have a website that plays with the user. The site acts as a kind of canvas, and each element that lives on it can be played with or repositioned. The user can create their own collage of shapes on either a black or white background. There aren’t any user prompts until some time passes; we want the user to explore the page and figure out what each piece does on their own.
