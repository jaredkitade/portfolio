Concept Lecture Series

Web Design

This website was created to promote the 2015 Concept Lecture Series for California College of the Art's Graphic Design program.

By using Paper.js and SVG files, the user is able to drag and rearrange the anchor points of the phrase "Concept Lecture Series." The website then uses an Optical Character Recognition (OCR) script to output what it sees.

With Danny Skitsko and Jack Koloskus.
